About the Project

Project Created By: Mohit

Project Title: Clinic Management System

This project is a secure, role-based healthcare management web application developed using HTML, CSS, JavaScript, and Firebase. It is designed to streamline clinical workflows by offering separate portals for receptionists and doctors.

Features:

  • Receptionist Role: Assign tokens, add new patient details, and manage queues.
  • Doctor Role: View patient information and add prescriptions.
  • Secure login and signup using Firebase Authentication.
  • Real-time data storage and access via Firebase Firestore.
  • Session management with logout confirmation.
  • Responsive design for various devices.

How It Works:

  • Users select their role and either log in or sign up.
  • Receptionists enter patient data which is stored in Firestore.
  • Doctors can view these entries and update prescriptions.
  • The data is synchronized and protected via Firebase‚Äôs real-time capabilities and security rules.
